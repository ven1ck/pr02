# Генератор QR-кодов

## Описание
Веб-приложение для оперативной генерации двумерных штрихкодов (QR-кодов) на основе произвольных текстовых данных. Сервис предоставляет интерфейс для настройки визуальных параметров изображения, интеграции пользовательского логотипа, систему аутентификации и ведение истории генераций для зарегистрированных пользователей.

## Основные возможности
- Генерация QR-кодов из URL, контактных данных (vCard), обычного текста
- Настройка размера изображения (200–600 пикселей)
- Выбор уровня коррекции ошибок: Низкий, Средний, Квартиль, Высокий
- Кастомизация цветовой схемы (foreground/background)
- Вставка логотипа в центр кода с автоматическим масштабированием
- Регистрация и авторизация пользователей
- История генераций с возможностью повторного скачивания файлов
- Административная панель для управления пользователями и архивом кодов
- Адаптивный интерфейс на Bootstrap 5 с переключением светлой/тёмной темы
- Оптимизация форм для мобильных устройств (крупные элементы ввода, корректные клавиатуры)

## Стек технологий
- **Backend:** Python 3.10+, Django 4.2+
- **Frontend:** Bootstrap 5.3, vanilla JavaScript, CSS
- **Генерация изображений:** `qrcode`, `Pillow`
- **База данных:** SQLite (по умолчанию)
- **Сборка:** стандартный `manage.py`

## Установка и запуск

1. Перейдите в корневую директорию проекта.
2. Создайте и активируйте виртуальное окружение:
   ```bash
   python -m venv .venv
   # Windows
   .venv\Scripts\activate
   # Linux/macOS
   source .venv/bin/activate
   ```
3. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
   или
    ```powershell
    pip install "Django>=4.2,<5.0" "qrcode[pil]>=7.4.2" "Pillow>=10.0.0" --index-url https://pypi.tuna.tsinghua.edu.cn/simple/ --trusted-host pypi.tuna.tsinghua.edu.cn
    ```
4. Примените миграции базы данных:
   ```bash
   python manage.py migrate
   ```
5. Создайте учётную запись администратора:
   ```bash
   python manage.py createsuperuser
   ```
6. Запустите сервер разработки:
   ```bash
   python manage.py runserver
   ```
7. Откройте в браузере: `http://127.0.0.1:8000/`

## Использование

### Генерация кода
- Введите текст или ссылку в основное поле.
- Укажите размер, уровень коррекции ошибок и цвета.
- При необходимости загрузите изображение логотипа.
- Нажмите кнопку `Сгенерировать QR-код`. Результат отобразится справа от формы.
- Используйте кнопку `Скачать PNG` для сохранения файла.

### Работа с учётными записями
- Незарегистрированные пользователи могут генерировать коды без ограничений.
- Зарегистрированные пользователи получают доступ к разделу `История`, где хранятся все их генерации с метаданными и ссылками для скачивания.
- Вход/выход и регистрация доступны через навигационную панель.

### Административная панель
- Доступ: `http://127.0.0.1:8000/admin/`
- Позволяет просматривать, фильтровать и удалять записи QR-кодов.
- Включает упрощённый интерфейс создания пользователей с отключённой строгой валидацией паролей.
